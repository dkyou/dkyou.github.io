8. initramfs /init 与 /sbin/init 的区别

两者容易混淆，但位置和角色不同。

通常位于 initramfs 解包出来的内存 rootfs 中。它是内核优先尝试执行的早期用户态程序。

通常位于真实根文件系统中，例如 eMMC、SD、UBIFS、NFS root 等。它是在真实根挂载完成后执行的系统 init。
